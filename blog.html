Question-1: What are the different ways to select an element in the DOM?

  1. document.getElementById("id-name")
  2. document.getElementByClassName("class-name")
  3. document.getElementByTagName("tag-name")
  4. document.querySelector(".class-name")
  5. document.querySelectorAll("tagName.className")

Question-2: What is the difference between innerHTML, innerText, and textContent ?

  1. innerHTML -> returns HTML tag and inner text of an element.
  2. innerText -> returns only the inner text of an element, without spacing and css hidden text.
  3. innerContent -> The text content of the element and all descendaces with spacing and css hidden text.

Question-3: What is event delegation in the DOM?

Event Delegation is a technique in JavaScript where you delegate the handling of events to a parent element rather than attaching event listeners to individual child elements.
